# Python租房系统 python505

本项目为Python语言编写的租房系统，采用Django框架后端结合Vue2前端，搭配Mysql数据库，实现了一整套房屋租赁管理的功能。

## 技术栈

- 后端：Python + Django
- 前端：Vue2
- 数据库：Mysql

## 功能模块

系统包括但不限于以下功能模块：

- 房源信息管理
- 租赁合同管理
- 用户权限管理
- 租金支付管理
- 数据统计与分析

## 系统角色

- 系统管理员
- 房东
- 租客

## 运行环境

- Python 3.x
- Django 2.x
- Vue 2.x
- Mysql 5.x

## 部署步骤

1. 准备Python及Django运行环境
2. 安装Mysql数据库，并进行配置
3. 克隆或下载项目源码至本地
4. 执行数据库迁移命令，以创建数据表
5. 运行前端和后端服务
